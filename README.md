Project Overview
This project provides a comprehensive analysis of chocolate sales data across different regions, products, and sales personnel. It transforms raw transactional data into actionable insights using data visualization techniques. The dashboard is designed to help stakeholders track performance, identify top-selling products, and monitor regional sales trends.

Key Features
KPI Tracking: Total revenue at a glance (63.95k).

Sales by Country: A horizontal bar chart showing performance across Australia, USA, India, Canada, New Zealand, and the UK.

Product Distribution: A donut chart detailing the revenue share of various chocolate types (e.g., Milk Bars, Organic Choco Syrup, Drinking Coco).

Salesperson Ranking: A combined bar and line chart visualizing the performance of top sales representatives and the volume of boxes shipped.

Dynamic Filtering: Ability to filter data by specific sales persons (e.g., Kelci Walkden, Madelene Upcott) and countries.

Tools & Technologies
Excel / Power BI / Tableau: (Loyiha qaysi dasturda qilinganiga qarab birini tanlang) for data modeling and visualization.

Data Source: Raw sales data including Sales Person, Country, Product, Date, Amount, and Boxes Shipped.

Dataset Structure
The underlying data (see image_1241d8.png) contains the following fields:
| Field | Description |
| :--- | :--- |
| Sales Person | Name of the employee who made the sale |
| Country | Region where the sale occurred |
| Product | The specific chocolate product sold |
| Date | Transaction date |
| Amount | Revenue generated from the sale |
| Boxes Shipped | Quantity of units delivered |

Insights Extracted
Top Region: Australia leads the sales performance among all countries.

Product Leader: "Milk Bars" and "Organic Choco Syrup" are the highest revenue contributors.

Top Performer: Karlen McCaffrey shows the highest shipment volume and ranking according to the performance metrics.
